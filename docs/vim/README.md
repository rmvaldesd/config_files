# Neovim docs

The comprehensive, maintained guide lives with the Neovim config itself so it
stays next to the code it documents and is reachable from `:help`:

- **[Full setup & usage guide](../../dotconfig/nvim/GUIDE.md)**
  (`dotconfig/nvim/GUIDE.md`) — prerequisites, debugging Go/Python/Django,
  navigation, and troubleshooting.

## Inside Neovim

- `:help lsp-nav` — code-navigation cheatsheet (jump to/from definitions, call
  hierarchy, symbols, references). Source: `dotconfig/nvim/doc/lsp-navigation.txt`.
- `<leader>?` — which-key popup of the current buffer's mappings.
- `<leader>km` — Telescope: fuzzy-search every keymap by description.
